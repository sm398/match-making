/game/assign

/game/assign post

post

post /game/assign

  • Request
  • Response

Headers

  • facebook_user_access_token: required (string)

    user access token given from facebook. Should be able to access atleast id and email of the facebook user.

  • gameType: required (string)
  • longitude: required (number)

    longitude of the location of player

  • latitude: required (number)

    latitude of the location of player

  • radius: required (boolean)

    radius in which the player wants to find the game (meters)

  • age: required (number)
  • reference: (string)

    If this user wants to join a game with another user. Enter the facebook id of the other user.

HTTP status code 200

The user has been added to the queue

HTTP status code 404

facebook user not found

HTTP status code 409

User already searching or playing a game

/game

/game get

get

get /game

  • Request
  • Response

Headers

  • facebook_user_access_token: required (string)

    user access token given from facebook. Should be able to access atleast id and email of the facebook user.

HTTP status code 200

After getting the response, the user is not queued for a game anymore

Body

Media type: application/json

Type: object

Properties
  • id: required (string)
  • gameType: required (string)
  • time: required (datetime)
  • place: required (object)
    • longitude: required (number)
    • latitude: required (number)
  • players: required (string)
  • ready: required (string)

    Boolean to flag if the game is complete and ready to be played. Once the game is ready, no more players will be added

HTTP status code 404

Facebook user is not found
